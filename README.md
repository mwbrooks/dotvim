VIM Configuration
=================

Install
-------

    # Download

    git clone git://github.com/mwbrooks/dotvim.git
    cd dotvim/

    # Deal with submodules

    git submodule init
    git submodule update

    # Install

    ln -s /path/to/dotvim ~/.vim
    ln -s /path/to/dotvim/vimrc ~/.vimrc
    ln -s /path/to/dotvim/gvimrc ~/.gvimrc
